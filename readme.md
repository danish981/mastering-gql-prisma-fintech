## ğŸš€ Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/mastering-gql-prisma-fin.git
cd mastering-gql-prisma-fin
```

### 2. Install Dependencies
```bash
npm install
# or
yarn install
```

### 3. Setup Environment
```bash
cp .env.example .env
```
Update `.env` with your PostgreSQL connection string:
```
DATABASE_URL="postgresql://username:password@localhost:5432/mastering_gql_prisma?schema=public"
```

### 4. Database Setup
```bash
npm run prisma:migrate  # Apply database migrations
npm run prisma:seed     # Seed initial data (optional)
```

### 5. Start Development Server
```bash
npm run dev
```
Server will start at `http://localhost:4000/graphql`

### 6. Explore with Prisma Studio
```bash
npm run prisma:studio
```
Open `http://localhost:5555` to visually explore your database.
